= Консольные программы. Обработка данных, введённых с клавиатуры. Форматированный вывод данных. Организация ветвления в программе

== Задание 1. Точный калькулятор делимости

*Условие:* Пользователь вводит одно целое число (делимое) и одно целое число (делитель). Гарантируется, что делитель не равен нулю.

*Задание:* Используя операции целочисленного деления (//) и нахождения остатка (%), выведите на экран фразу:
"Число А = <делимое>. Результат деления на В: <результат>, остаток: <остаток>."

*Пример:* Для ввода 10 и 3, вывод: "Число А = 10. Результат деления на В: 3, остаток: 1."

== Задание 2. Конвертер валют с большой точностью

*Условие:* Пользователь вводит сумму в рублях (может быть дробной). Курс доллара фиксирован и равен 95.50 руб.

*Задание:* Вычислите эквивалент в долларах с точностью до двух знаков после запятой, используйте метод quantize(), и выведите результат.

*Пример:* Для ввода 1000, вывод: "1000.00 рублей = 10.47 долларов" (1000 / 95.50 ≈ 10.4712 -> 10.47).

== Задание 3. Маскировка номера карты

*Условие:* Пользователь вводит номер банковской карты в виде строки из 16 цифр (например, 1234567890123456).

*Задание:* С помощью срезов строк сформируйте и выведите номер в замаскированном виде, показывая только последние 4 цифры. Остальные цифры замените на `*`. 

Формат вывода: `**** **** **** 3456` (группы по 4 символа через пробел).

*Пример:* Для ввода 1234567890123456, вывод: "`**** **** **** 3456`"

== Задание 4. Работа с кортежами и распаковкой

*Условие:* Функция возвращает кортеж с данными пользователя:
user_data = ("Смирнова", "Екатерина", 28, "Москва")

*Задание:* Распакуйте этот кортеж в отдельные переменные: surname, name, age, city. Выведите красочно отформатированную строку с их использованием.

*Пример вывода:* "Сотрудник: Смирнова Екатерина, возраст: 28 лет, город: Москва".

== Задание 5. Инвентарь магазина

*Условие:* Дан словарь, описывающий остатки товаров на складе.
inventory = {"яблоки": 150, "бананы": 200, "апельсины": 75, "киви": 0, "груши": 30}

*Задание:* Пришла поставка: 50 яблок, 30 бананов и 20 груш. Обновите словарь inventory, используя метод update().

*Пример:* После обновления для ключа "яблоки" должно быть значение 200.

== Задание 5. Преобразование типов при вводе
*Условие:* Пользователь вводит три значения: свое имя (строка), год рождения (целое число) и средний балл (дробное число) — в одну строку, разделяя их пробелом.

*Задание:*

* Считайте строку и используйте метод split() для разделения.
* Сохраните имя как есть.
* Год рождения преобразуйте в int.
* Средний балл преобразуйте в float.
* Вычислите приблизительный возраст пользователя, взяв за текущий год 2026.
* Выведите итоговую информацию.
Пример: Для ввода "Анна 2010 4.75", вывод: "Студентка: Анна, Возраст: 16 лет, Средний балл: 4.75"
