= Классы

== Задача 1. Класс «Студент»

*Создайте класс Student, который хранит информацию о студенте.*

Требования:
. Атрибуты: `name (имя), group (группа), grades (список оценок)`

. Метод `add_grade(grade)` — добавляет оценку в список

. Метод `average_grade()` — возвращает средний балл

. Метод `info()` — выводит информацию о студенте

*Пример*
```py
student = Student("Иван Иванов", "ПИБ-101")
student.add_grade(5)
student.add_grade(4)
student.add_grade(5)
print(student.average_grade())  # 4.67
student.info()  # Студент Иван Иванов, группа ПИБ-101, средний балл: 4.67
```

== Задача 2. Класс «Сотрудник»

*Создайте класс Employee, который хранит информацию о сотрудниках компании.*

Требования:
. Атрибуты: name, position, salary

. Приватный атрибут __email (формируется автоматически: имя.фамилия@company.com)

. Свойство salary с сеттером (зарплата не может быть меньше 0)

. Метод raise_salary(percent) — повышение зарплаты

. Статический метод validate_name(name) — проверка имени (только буквы)

. Классовый метод from_string(data) — создание сотрудника из строки "Имя,Должность,Зарплата"

*Пример*
```py
emp = Employee("Иван Петров", "Разработчик", 50000)
emp.raise_salary(10)
print(emp.salary)  # 55000
emp2 = Employee.from_string("Анна Сидорова,Тестировщик,45000")
```
