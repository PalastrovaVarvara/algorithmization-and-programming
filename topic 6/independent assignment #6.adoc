= Файлы

== Задача 1. Поиск и замена в текстовом файле

*Напишите программу, которая:*

. Считывает файл document.txt.
. Подсчитывает количество упоминаний слова "дождь" (регистр игнорируется).
. Заменяет все упоминания слова "дождь" на "ливень".
. Создаёт обновлённый файл document_updated.txt с внесёнными изменениями.
. Выводит на экран количество заменённых слов.

_Содержание файла document.txt_
```txt
Сегодня было солнечно. Завтра обещают дождь.
Погода этой осенью удивительно тёплая.
Вчера шёл дождь, а теперь снова ясно.
```

== Задача 2. Слияние файлов

У нас есть два текстовых файла:

_students.txt_ (содержит список студентов)
```txt
Иван Иванов
Анна Смирнова
Петр Кузнецов
```
_scores.txt_ (содержит их оценки)
```txt
Иван Иванов, 85
Анна Смирнова, 92
Петр Кузнецов, 78
```
*Необходимо:*

* Объединить данные из двух файлов.
* Создать файл report.txt с результатами в формате:
```txt
Иван Иванов: 85 баллов
Анна Смирнова: 92 балла
Петр Кузнецов: 78 баллов
Средний балл: 85 баллов
```

== Задача 3. Подсчёт строк в файле

Напишите программу, которая подсчитывает количество строк, слов и символов в текстовом файле

Пример
```txt
Имя файла: text.txt
Количество строк: 15
Количество слов: 234
Количество символов: 1256
```

== Задача 4. Телефонный справочник

Создайте программу для ведения телефонного справочника. Данные хранятся в файле `contacts.txt` в формате: Имя,Телефон. Реализуйте следующие функции:

. `add_contact(name, phone)` — добавляет новый контакт
. `search_contact(name)` — ищет контакт по имени и выводит номер
. `delete_contact(name)` — удаляет контакт
. `show_all()` — выводит все контакты
. `menu()` — текстовое меню для взаимодействия с пользователем

Пример работы:
```txt
1. Добавить контакт
2. Найти контакт
3. Удалить контакт
4. Показать все контакты
5. Выход
Выберите действие: 1
Введите имя: Анна
Введите телефон: 89123456789
Контакт добавлен!
```

== Задача 5. Шифрование файла

Напишите программу, которая шифрует и расшифровывает текстовый файл с помощью шифра Цезаря (сдвиг букв на определённое число позиций).

Требования:

. Функция `encrypt_file(input_file, output_file, shift)` — шифрует файл
. Функция `decrypt_file(input_file, output_file, shift)` — расшифровывает файл
. Сохраняйте только русские и английские буквы, остальные символы оставляйте без изменений

Пример:

```txt
Исходный файл: "Привет, мир!"
Шифр (сдвиг 3): "Тулёз, плу!"
Расшифровка: "Привет, мир!"
```
