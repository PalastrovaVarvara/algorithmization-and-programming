= Условные операторы (Ветвление).

== Задание 1. Определение времени суток

*Задание:* Напишите программу, которая запрашивает у пользователя текущий час (от 0 до 23) и выводит соответствующее сообщение:

* 6-11: "Доброе утро!"
* 12-17: "Добрый день!"
* 18-23: "Добрый вечер!"
* 0-5: "Доброй ночи!"

*Входные данные:* целое число от 0 до 23

*Выходные данные:* строка с приветствием

*Пример:* 
....
Введите час: 14
Добрый день!
....

*Критерии проверки:*

* Корректная обработка всех диапазонов
* Обработка некорректного ввода (числа вне диапазона)

== Задание 2. Калькулятор скидки

*Задание:* Магазин предоставляет скидки в зависимости от суммы покупки:

* До 1000 руб. — скидка 0%
* От 1000 до 3000 руб. — скидка 5%
* От 3000 до 5000 руб. — скидка 10%
* Свыше 5000 руб. — скидка 15%

Напишите программу, которая запрашивает сумму покупки и выводит итоговую сумму с учётом скидки.

*Входные данные:* вещественное число (сумма покупки)

*Выходные данные:* сумма к оплате с округлением до 2 знаков

*Пример:*
....
Введите сумму покупки: 3500
Скидка: 10%
Итоговая сумма: 3150.00 руб.
....

*Дополнительное задание:* Добавьте проверку, что сумма не может быть отрицательной.

== Задание 3. Определение типа треугольника

*Задание:* Напишите программу, которая запрашивает длины трёх сторон треугольника и определяет его тип:

* Равносторонний (все стороны равны)
* Равнобедренный (две стороны равны)
* Разносторонний (все стороны разные)

Дополнительно проверьте, существует ли треугольник с такими сторонами (неравенство треугольника: сумма любых двух сторон должна быть больше третьей).

*Входные данные:* три целых или вещественных числа

*Выходные данные:* тип треугольника или сообщение, что треугольник не существует

*Пример:* 

....
Введите сторону a: 5
Введите сторону b: 5
Введите сторону c: 8
Тип треугольника: равнобедренный
....

*Подсказка:* Для проверки существования треугольника нужно проверить все три неравенства.

== Задание 4. Проверка пароля

*Задание:* Напишите программу, которая запрашивает у пользователя логин и пароль. Сравните их с заранее заданными значениями (например, логин: "admin", пароль: "12345").

* Если логин и пароль верны → "Доступ разрешён"
* Если логин верен, а пароль нет → "Неверный пароль"
* Если логин неверен → "Пользователь не найден"

*Входные данные:* две строки (логин и пароль)

*Выходные данные:* сообщение о результате проверки

== Задание 5. Обработка команд

*Задание:* Напишите программу-интерпретатор команд. Пользователь вводит команду в виде строки, программа разбирает её и выполняет соответствующее действие.

*Поддерживаемые команды:*

* hello → вывести "Привет, мир!"
* hello <имя> → вывести "Привет, <имя>!"
* add <x> <y> → вывести сумму x и y
* sub <x> <y> → вывести разность x и y
* mult <x> <y> → вывести произведение x и y
* div <x> <y> → вывести частное x и y (с проверкой деления на 0)
* info → вывести информацию о программе
* exit → завершить программу

*Пример работы:*

```
>> hello Анна
Привет, Анна!
>> add 5 3
Сумма: 8
>> exit
Программа завершена.
```

Подсказка: Используйте match с шаблонами последовательностей:

``` python
match command.split():
    case ["hello"]:
        print("Привет, мир!")
    case ["hello", name]:
        print(f"Привет, {name}!")
    case ["add", x, y]:
        print(f"Сумма: {int(x) + int(y)}")
```
