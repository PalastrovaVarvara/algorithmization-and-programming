== Задание 1. Конвертер температур

*Задание:* 

Создайте набор функций для конвертации температур:

. `celsius_to_fahrenheit(c)` — переводит из Цельсия в Фаренгейты
. `fahrenheit_to_celsius(f)` — переводит из Фаренгейтов в Цельсий
. `celsius_to_kelvin(c)` — переводит из Цельсия в Кельвины
. `kelvin_to_celsius(k)` — переводит из Кельвинов в Цельсий

Формулы:

. `°F = °C × 9/5 + 32`
. `°C = (°F - 32) × 5/9`
. `K = °C + 273.15`
. `°C = K - 273.15`

*Пример:*

....
print(celsius_to_fahrenheit(25))    # 77.0
print(fahrenheit_to_celsius(77))    # 25.0
print(celsius_to_kelvin(0))         # 273.15
print(kelvin_to_celsius(273.15))    # 0.0
....

*Дополнительное задание:* Напишите функцию `convert_temperature(value, from_unit, to_unit)`, которая принимает значение, исходную единицу и целевую единицу и выполняет преобразование, используя созданные функции.

== Задание 2. Имитация банковского счёта

*Задание:* 

Реализуйте банковский счёт с помощью функций и глобальной переменной.

Требования:
Глобальная переменная balance хранит текущий баланс

. Функция `deposit(amount)` — пополняет счёт
. Функция `withdraw(amount)` — снимает деньги (проверяет, достаточно ли средств)
. Функция `get_balance()` — возвращает текущий баланс
. Функция `reset_balance()` — сбрасывает баланс до 0

*Пример:*

....
deposit(1000)
deposit(500)
withdraw(300)
print(get_balance())  # 1200
withdraw(2000)        # Недостаточно средств!
reset_balance()
print(get_balance())  # 0
....

*Дополнительное задание:* Добавьте защиту от прямого изменения глобальной переменной вне функций (через документацию и инкапсуляцию).

== Задание 4. Проверка палиндрома

Напишите рекурсивную функцию `is_palindrome(s)`, которая проверяет, является ли строка палиндромом (читается одинаково слева направо и справа налево). Игнорируйте пробелы, знаки препинания и регистр.

*Пример:*

....
print(is_palindrome("А роза упала на лапу Азора"))  # True
print(is_palindrome("Hello"))  # False
print(is_palindrome("12321"))  # True
....

*Подсказка:* Для удаления пробелов и приведения к одному регистру используйте строковые методы.
